FCM Web App

This web app should run on PHP web server or local server by placing the files in your web directory.

If it runs on remote PHP web server, the app should be accessed trough https protocol.

To initialize the web app, go to Setup page, and specify the required information which can be taken from this link.

The VAPID key can be taken from the Web Push certificates from this link.

Click the Initialize button to initialize the FCM Web Client.

After the registration token was generated, you can now use for the registeration token in your Arduino code.

In Windows 11 PC, the notification may go directly to the notification center instead of displaying pop-up, if Do not disturb is On in the System:Notifications and you have to turn it Off to allow the notification pop-up to show.

Some additional credentials are required for OAUth2.0 authorization when sending and subscribing to the topic which are taken from the Service Account.
